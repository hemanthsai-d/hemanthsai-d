## 🔭 What I Focus On

I'm an **impact-driven Data Scientist with 5 years of experience** delivering production-grade machine learning, forecasting, and real-time analytics solutions at scale. Here's what drives my work:

- 🧠 **End-to-End ML Systems** — From data ingestion to model deployment, I build the full pipeline
- 📈 **Time-Series Forecasting** — ARIMA, SARIMA, Prophet, XGBoost, LightGBM for demand prediction
- ⚡ **Real-Time Analytics** — Kafka + Spark Streaming for sub-second anomaly detection
- 🔄 **MLOps & Automation** — CI/CD retraining, model monitoring, automated pipelines with Airflow & SageMaker
- 🤖 **AI Engineering** — RAG systems, LLM orchestration, calibrated confidence scoring, guardrails
- ☁️ **Cloud-Native Architecture** — AWS-first design with cost optimization and infrastructure-as-code
